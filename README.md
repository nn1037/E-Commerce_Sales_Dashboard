#E-commerce Sale Dashboard
Description
This Power BI project is an interactive Online Sales Dashboard that provides insights into sales performance across various product categories. It enables users to track metrics such as sales amount, profit, and quantity sold, while also offering filters based on customer details and regions.

The dashboard is designed to help stakeholders and decision-makers easily monitor sales performance and identify opportunities for growth.

Features
Sales Overview: Track total sales amount, profit margins, and quantities sold across different product categories.
Customer Insights: Explore customer purchase behaviors by analyzing sales data based on customer locations (State, City) and payment modes.
Time-based Analysis: The dashboard allows for filtering and analysis of sales trends over specific periods.
Interactive Filters: Users can drill down into specific categories such as Electronics, Furniture, or different payment modes (COD, EMI, Credit Card).
Data
The project uses two datasets:

Details.csv: Contains information about individual orders, including:

Order ID: Unique identifier for each order.
Amount: Total sales amount for each order.
Profit: Profit made from the sale.
Quantity: Quantity of items sold.
Category and Sub-Category: Product categorization.
Payment Mode: Payment method (COD, EMI, Credit Card).
Orders.csv: Contains customer and order details:

Order ID: Unique identifier for each order.
Order Date: The date the order was placed.
Customer Name: Name of the customer.
State, City: Location of the customer.
Setup and Installation
Clone the repository:
git clone https://github.com/your-username/online-sales-dashboard.git
Open the .pbix file in Power BI Desktop.
Usage Once the dashboard is loaded in Power BI, you can:
Filter data based on Order Date, State, City, or Category. Analyze sales trends, profits, and quantities sold. Drill down into specific customer segments or product categories.
